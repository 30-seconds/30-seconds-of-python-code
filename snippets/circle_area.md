Finds the area of circle from radius

- Uses the formula `area of circle = pi x r x r`

```py
import math

def circle_area(radius):
  return math.pi * radius * radius
```

```py
circle_area(2) # 12.566370614359172
circle_area(3.14159) # 31.00622430055893
```
