Converts  Kelvin to Fahrenheit.

- Follow the conversion formula `F = 9/5 * (K - 273) + 32`.

```py
def kelvin_to_fahrenheit(degrees):
  return ((9/5)*(degrees - 273) + 32)
```

```py
kelvin_to_fahrenheit(273) # 32.0
```
