Converts Kelvin to Fahrenheit.

- Using formula `fahrenheit = ((kelvin - 273.15) * (9/5) + 32)`.

```py
def kelvin_to_fahrenheit(kelvin):
  return ((kelvin - 273.15) * (9/5) + 32)
```

```py
kelvin_to_fahrenheit(50) # -369.66999999999996
```
