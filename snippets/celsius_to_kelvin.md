Converts temperature from Celsius to Kelvin.
- Follows the conversion formula: `K = C + 273.15`.

```py
def celsius_to_kelvin(temp_input):
    return temp_input + 273.15
```

```py
celsius_to_kelvin(32) #305.15
```
